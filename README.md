# from_scores_to_decision
Data and codes for "From Scores to Decision: Unveiling Company Maturity for Industry 4.0 Using Benefit-of-the-Doubt Models"

# Reproducibility Materials

This repository contains the data and code used in the study.

## Data
- `data/Data.xlsx`: anonymized dataset used in the analysis

## R Code (BoD Analysis)
- `R/BoD_analysis.R`
- Computes composite indicators using the BoD approach

## Python Code (GAM Analysis)
- `Python/GAM_analysis.ipynb`
- Implements GAM models and generates plots

## How to Run

1. Open the Excel file in the `data/` folder
2. Run the R script for BoD analysis
3. Open the Python notebook in Google Colab:
   - Upload the dataset when prompted
   - Run all cells

## Notes
- All code was tested before submission
- Results in the paper can be reproduced following the steps above
