# from_scores_to_decision
Data and codes for "From Scores to Decision: Unveiling Company Maturity for Industry 4.0 Using Benefit-of-the-Doubt Models"

# Reproducibility Materials

This repository contains the data and code used in the study.

## Data
- `Data.xlsx`: anonymized dataset used in the analysis

## R Code (BoD Analysis)
- `R_code_for_composite_indicator_determination.R`
- Computes composite indicators using the BoD approach

## Python Code (GAM Analysis)
- `Python_code_bod_paper_gams.ipynb`
- Implements GAM models and generates plots
